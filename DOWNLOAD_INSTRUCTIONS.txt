QUICK START (after getting files)
================================================================================

Windows:
--------
1. Double-click START.bat
2. Open browser to http://localhost:3333
3. Login: admin / admin

Mac/Linux:
----------
1. Open Terminal in the phishSimAI folder
2. Run: chmod +x START.sh
3. Run: ./START.sh
4. Open browser to http://localhost:3333
5. Login: admin / admin

Manual Start:
-------------
1. Open Terminal/Command Prompt in phishSimAI folder
2. Run: go mod download
3. Run: go run main.go
4. Open browser to http://localhost:3333
5. Login: admin / admin
